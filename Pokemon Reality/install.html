Comment installer Pokemon Reality pour la première fois ?

Bonjour, si vous êtes sur cette page, c'est que vous cherchez à installer pour la première fois Pokemon Reality ! Installez-vous donc bien et commençons...

Tutoriel écrit :

1ère étape : Extraire le fichier "Pokemon Reality (alpha/beta) x.x.x.rar" sur votre ordinateur.


2ème étape : Vous avez maintenant un fichier "Pokemon Reality (alpha/beta) x.x.x.rar" et un dossier "Pokemon Reality (alpha/beta) x.x.x"

3ème étape : Supprimez le fichier "Pokemon Reality (alpha/beta) x.x.x.rar"

4ème étape : Lancez votre jeu en lançant "Game.exe" dans le dossier "Pokemon Reality (alpha/beta) x.x.x"

5ème étape : Amusez-vous !
