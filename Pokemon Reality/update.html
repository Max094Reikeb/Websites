Comment mettre à jour mon jeu sans perdre ma sauvegarde ?

Bonjour, vous venez de télécharger la dernière version en date de Pokemon Reality et vous ne savez pas comment mettre à jour sans perdre votre sauvegarde ? Ce mini-guide est pour vous !


Tutoriel écrit :

1ère étape : Allez dans un Centre Pokémon ! Sauf si la version ne l'indique, vous devez aller dans un Centre Pokémon afin de ne pas changer de location si la map se retrouve modifiée. Allez donc soigner vos Pokémon un petit coup !

2ème étape : Sauvegardez et éteignez votre jeu...

3ème étape : Dans le dossier de votre jeu "Pokemon Reality (alpha/beta) x.x.x", localisez le dossier "Saves"

4ème étape : Dans ce dossier, copiez le fichier "Pokemon_Party" qui contient donc votre sauvegarde.

5ème étape : Extrayez le dossier "Pokemon Reality (alpha/beta) x.x.x" du fichier ".rar" que vous avez téléchargé.

6ème étape : Cherchez dans ce nouveau dossier le dossier "Saves" qui est vide (normalement !)

7ème étape : Y coller le fichier "Pokemon_Party" précédemment copié à l'étape 4.

8ème étape : Supprimez le dossier "Pokemon Reality (alpha/beta) x.x.x" des étapes 3 et 4.

9ème étape : Lancez votre "Game.exe" du nouveau dossier précédemment téléchargé et jouez à la nouvelle version !
